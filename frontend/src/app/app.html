Session 列表是空的，怎麼辦？
確認「Claude Code 專案目錄」設定是否正確（⚙ 設定）。
Claude Code 的對話記錄存在 ~/.claude/sessions/，若從未使用過 CLI，該目錄可能是空的。
Agents / Skills 頁籤顯示空白？
需要在對應目錄建立 .md 檔案：
~/.claude/agents/ — 放 Agent 檔案
~/.claude/skills/ — 放 Skill 檔案
建立後點右上角 ↻ 重新整理。
傳送訊息出現認證錯誤？

「專案目錄」要填什麼？
填你平常在終端機 cd 到的專案根目錄的絕對路徑。

這個路徑決定從 ~/.claude/projects/ 的哪個子目錄讀取資料。

如何重新開啟設定精靈？
進入 ⚙ 設定，滾動到最下方，點「🧭 精靈」按鈕。
後端連線中斷怎麼辦？
